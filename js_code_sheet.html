Function pprintToScreen

we use this function in js to show something on the screen, first we create
a console as text holder, then we write into it.
// This line to be in the HTML where we want the text to apear.   <div id="screen-console">Waiting for your input...</div>

     function printToScreen(text) {
            const consoleBox = document.getElementById('screen-console');
            if consoleBox.innerText= "Enter an item to added to the list." {
                consoleBox.innerText= "";
                 }
            else {
                consoleBox.innerText += text + "\n";
            }
        }
