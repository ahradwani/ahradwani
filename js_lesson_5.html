JavaScript Lesson 5: The List

Create, edit the list in JS.

Common List OperationsJavaScript provides built-in methods to easily change or query your list:
create a list: things=["item1","item2","item3",..]

Add to end of the list: fruits.push("mango");
adds an item to the very bottom of the list.

Add to start: fruits.unshift("apricot");
inserts an item at the beginning.

Remove from end: let lastItem = fruits.pop();
removes and returns the last element.

Remove from start: let firstItem = fruits.shift();
removes and returns the first element.

Get length: fruits.length;
counts the total number of entries in your list.

Check contents: fruits.includes("banana");
returns true or false based on item presence.
