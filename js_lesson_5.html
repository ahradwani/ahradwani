JavaScript Lesson 5: The List

Create, edit the list in JS.

Common List Operations: JavaScript provides built-in methods to easily change your list:

Create a list: let things = ["item1", "item2"]
Add to end: things.push("item3");
Add to start: things.unshift("item4");
Remove from end: things.pop();
Remove from start: things.shift();
