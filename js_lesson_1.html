The Concept In programming, you need a way to remember data. Think of a Variable as a labeled storage box. We use let to create a box whose contents can change later (like a score in a game).
We use const to create a box whose contents are locked and can never change (like your name or a fixed setting).

In this lesson we will learn about the let and const. We can store a value in the let as let x = 5, let y="Qatar", let Myname="Coco Wawa".
The value in the let variable cna be changed during out program running time.
So if we have let score = 5, then we can say score += 1.

Values in const will can't be cchanged, so if we name Myname = 'Coco Wawa' then it will not chabge.
