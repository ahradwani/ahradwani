Cascading Style Sheets (CSS)

CSS (Cascading Style Sheets) is the language used to style and layout web pages. While HTML provides the structure and content of a page, CSS provides the visual presentation—controlling colors, fonts, spacing, and responsive design.

Why We Use CSS

  • Separation of Concerns: Keeps content (HTML) separate from presentation (CSS), making code easier to maintain.
  • Efficiency: You can apply the same style rules across multiple pages from a single file.
  • Responsive Design: Allows websites to adapt gracefully to different screen sizes (mobile, tablet, desktop).
  • Consistency: Ensures a uniform look and feel across your entire website.

Essential CSS Properties

These are the fundamental properties every developer uses to build layouts:

  • color: Sets the text color.
  • background-color: Sets the background color of an element.
  • font-family: Defines the typography/font choice.
  • margin: Creates space outside an element's border.
  • padding: Creates space inside an element, between the content and the border.
  • display: Controls how elements are rendered (e.g., block, inline, flex).

Advanced Properties for Better Layouts

To move beyond basic styling, incorporate these powerful tools:

  • Flexbox (display: flex): Ideal for aligning items in a row or column and distributing space.
  • CSS Grid (display: grid): Used for complex, two-dimensional layouts.
  • Media Queries (@media): Allows you to apply specific styles based on the device's width.
  • Transitions/Animations: Adds fluid movement and interaction to elements.

Linking CSS to HTML

The best practice is to use an external stylesheet. Place a <link> tag inside the <head> section of your HTML document:

<head>
    <link rel="stylesheet" href="style.css">
</head>
