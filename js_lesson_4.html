JavaScript Learning: for loop in js

A for loop in JavaScript is a control flow statement that allows
you to repeatedly execute a block of code a specific number of times.
It is most commonly used when you know exactly how many times you need to iterate.

Breakdown of Components Initialization: Executed once before the loop starts. Typically used to set up
a counter variable (e.g., let i = 0). Condition: Checked before each iteration. If it evaluates to true, the loop continues;
if false, the loop stops. Final-expression: Executed after each iteration. Typically used to increment or
decrement the counter (e.g., i++).
